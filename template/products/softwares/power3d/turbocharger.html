Turbocharger

Project example

A project that descibes a turbine-driven forced induction device fuctionality.
We integrated in the presentation 3D animated components(turbine wheel, compressor wheels, bearing system etc) and real-time simulations, showing how the pieces combine and the working flows (exhaust or air intel/outlet), cut-away views, floating parts, explosion effects, color highlighted cooling passages.
With proper upgrading, it can be used to create online presentation of any turbine-based system (engine, car ) or in exhibition set ups, together with 3D interactive models, lighting panesl, videowall or interactive hologram outputs.
